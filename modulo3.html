Módulo 3: Proteja seu dinheiro.

Coisas que o banco pode fazer:

  1. Monitoramento em tempo real: O banco utiliza sistemas de inteligência que analisam suas compras 24 horas por dia para identificar qualquer movimentação que não combine com o seu perfil.
  2. Bloqueio preventivo de segurança: Caso o banco detecte uma tentativa de compra em um local muito distante ou com um valor muito alto, ele pode bloquear o cartão imediatamente para evitar prejuízos.
  3. Confirmação por biometria facial: Para garantir que é realmente você, o banco utiliza o reconhecimento do seu rosto no aplicativo antes de autorizar qualquer transferência importante.
  4. Seguro contra transações indevidas: O banco oferece proteções específicas que podem cobrir valores caso você seja vítima de um golpe ou tenha o celular roubado.
  5. Criação de cartões digitais temporários: O sistema do banco gera números de cartões que expiram logo após o uso, garantindo que os dados do seu cartão físico nunca sejam expostos em sites.

Coisas que o banco não pode fazer:

  1. O banco nunca pede sua senha por telefone: Se alguém ligar pedindo os números que você usa para entrar na conta ou para pagar contas, desligue na hora. Essa informação é só sua.
  2. O banco nunca manda ninguém buscar o seu cartão: Mesmo que digam que o cartão foi "clonado" ou "está com problema", o banco jamais enviará um funcionário ou motoboy até a sua casa para recolhê-lo.
  3. O banco nunca pede para você transferir dinheiro para "segurança": Nenhum funcionário legítimo pedirá para você fazer um Pix ou transferência para uma "conta de teste" ou "conta de proteção". Isso não existe.
  4. O banco nunca solicita que você instale aplicativos estranhos: Eles jamais pedirão para você baixar um programa que permita que alguém controle o seu celular à distância ("acesso remoto").
  5. O banco nunca pede para você cancelar um golpe fazendo outro Pix: Se disserem que houve um erro e que você precisa fazer uma nova transferência para "estornar" o valor, é mentira.

💡 Dicas Extra:

1. Evite clicar em links de e-mails suspeitos.

2. Use um antivírus atualizado.

3. Mantenha seu sistema operacional e navegadores sempre atualizados.

💌 Um lembrete para o seu coração

Sua pressa é a aliada do erro: Se alguém te apressar, pare. Você tem o direito de levar o tempo que precisar para decidir.

Pedir ajuda é um ato de sabedoria: Conversar com um filho, neto ou amigo sobre uma dúvida não é incomodar, é compartilhar cuidado.
